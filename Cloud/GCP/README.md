- Computee Engine -> Criar Máquina Virtual
- Vertex AI -> Treinar e hopedar modelos de Machineee Learning
- Cloud SQL
- Big Query
- Kubernetes
- Text-to-Speech
- Speech-to-Text
- Cloud Vision
- Cloud Vision Intelligence
- Cloud Translation
- Cloud Run
- Cloud Storage
